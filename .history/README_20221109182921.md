# Installing & Setting Up Streamlit:
## Install Pipenv

```
python -m ensure pip --upgrade

pip3 install pipenv
```

## Create a new environment with Streamlit

```
cd myproject

pipenv shell

pipenv install streamlit

streamlit hello
```

## Use your new environment
```
pipenv shell
streamlit run myfile.py
```
